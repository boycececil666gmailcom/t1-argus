# Argus 👁

> *Named after Argus Panoptes — the hundred-eyed giant of Greek mythology who never slept and watched everything.*

A daemon that silently runs in the background, recording which apps and windows you actively use and for how long. Generates reports to help you understand your own habits and behavior patterns.

---

## What it does

- Tracks the active foreground window (app name, window title) at regular intervals
- Detects idle time (no keyboard/mouse input) and excludes it
- Persists data to a local database
- Generates daily/weekly reports: time per app, time per category, focus sessions, distraction patterns
